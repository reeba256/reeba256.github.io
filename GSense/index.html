G-Sense

Description:

     See how much you earn from Admob & Adsense. GSense provide a simple and easy platform to track your earnings and visualize them.

Features
  • Real time earnings
  • Compare with previous data
  • Select different date ranges
  • Visualize in chart
  • Select any currency
  • Select number of items in the chart
