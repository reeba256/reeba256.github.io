Indic Handwriting Keyboard

Description:

The first FREE handwriting keyboard for all Indian Languages with word suggestion in iOS, which can be accessed through out the device.

Languages

  • Hindi
  • Punjabi
  • Tamil
  • Marathi
  • Kannada
  • Gujarati
  • Malayalam
  • Telungu
  • Nepali
  • Bengali
  • Urdu
  • Oriya

    Features

  • Keyboard with prediction/google suggestions enabled.
  • Easy to switch between all available keyboards in the device by clicking the globe icon.


    This is a free app which lets user to use both keyboards as device's normal keyboard. Indic handwriting keyboard can be accessed everywhere in the device, across every applications, and NOT just for a particular application.
