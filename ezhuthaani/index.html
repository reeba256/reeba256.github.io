Description:

     The first FREE Malayalam, Manglish & Handwriting keyboard with predictions which can be accessed through out the device.

     This is a free app which lets user to use any of the 3 keyboards as device's normal keyboard. Ezhuthaani can be accessed everywhere in the device, across every applications, and NOT just for a particular application.

     There are 3 keyboards available in this app,
     Manglish keyboard : Which converts English to Malayalam.
     Handwriting Keyboard : Draw the words and convert them to letters
     Malayalam script keyboard : Where the keyboard displays malayalam letters as such.
     User has an option to enable any one, or both keyboards at the same time.

Features:

  • Manglish keyboard shows English alphabets in the keyboard and convert what you type to Malayalam.
  • Malayalam script keyboard shows malayalam layout.
  • Handwriting Keyboard
  • NO internet connection required.
  • Keyboard with prediction/suggestions enabled.
  • Use as a normal English keyboard or Manglish keyboard with out changing the keyboard. You can do it by just using a toggle button within the keyboard settings.
  • Easy to switch between all available keyboards in the device by clicking the globe icon.
  • Uses default iOS theme.
